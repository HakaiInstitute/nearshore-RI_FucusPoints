# Changelog

## [Unreleased]

## [3.1.0] 2024-01-10
### Changed

- Updated README file

### Changed
- Removed "taxon_code" columns from survey data and replaced with 
"scientific_name" and "description" columns.
- Updated survey data to November 2022

## [2.0.0] 2022-01-11
### Changed
- Updated survey data to November 2021

## [1.0.0] 2020-03-04
### Added
- Creative Commons license
- README file
- Overall Rocky Intertidal protocol
- Example R script for summarizing data
- Example figure for summarizing data

### Changed
- Updated survey spreadsheet with 2019 survey data
- Updated Data Dictionary
- Updated taxa spreadsheet with new codes and structure

- Fucus Bed Dynamics protocol

## [0.1.0] - 2019-02-15
### Added
- Fucus Bed Dynamics survey data
- Rocky Intertidal taxa codes
- Rocky Intertidal site data
- Data Dictionary with definitions of all variables
- Fucus Bed Dynamics protocol
